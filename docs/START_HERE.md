# Start Here

## What is Supabase?

Supabase is just the backend service we will use so you do not need to build servers from scratch.

For this app, we would use it for:

1. saving uploaded resume files
2. storing user history and purchases
3. running a small server function that calls OpenAI
4. handling login later if we need it

You do not need to understand it deeply right now.

## What works already

The app already works in `local preview mode`:

1. open the app in browser
2. choose a resume file
3. run analysis
4. see the paywall and result flow
5. saved scans appear on the home screen

This is enough to understand the MVP shape.
